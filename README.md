# CRM BDS (Real Estate CRM)

A lightweight CRM system for real estate businesses, built with Next.js and Google Sheets as a database.

---

## 🚀 Features

- Customer Management (KHACH_HANG)
- Sales Pipeline (PIPELINE)
- Task Management (CONG_VIEC)
- Staff Management (NHAN_VIEN)

---

## 🛠 Tech Stack

- Next.js (App Router)
- API Routes
- Google Sheets (Database)
- Deployment: Vercel
